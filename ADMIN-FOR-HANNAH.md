# FarmDrop Admin - Quick Start for Hannah

## Where to sign in

https://farmdrop.vercel.app/admin

Enter your password. That unlocks the editor for the whole current session.

## What you can edit

All the event text on the events page - titles, times, locations, descriptions, and button labels/links - plus the Food for Health section, the section intros, and the BCZ YapZ interview block.

Stats like "65+ farm partners", the logo, the brand colors, and the home page structure are still in code. Ping Zaal if any of those need a change.

## How to save

1. Edit any field in the form.
2. Click **Save and Publish** at the bottom.
3. You'll see "Saved. Changes are live now."
4. Refresh the events page - your changes are already there.

The farmdrop.us events embed reads the same data, so your edit shows up there too automatically.

## Tips

- Blank out a secondary button's label to hide it from the event card.
- Use full URLs on button links (`https://...` or `mailto:...`).
- If you want to see the latest saved version without losing your edits-in-progress, use the `Reload from live` button.

## Signing out

Click **Sign out** in the top right. Closing the tab also clears your session.

## Troubleshooting

- **"Wrong password"**: double check. Zaal can reset if needed.
- **"Save failed"**: network blip, try again. If it keeps failing, text Zaal.
- **Your edit didn't show up on the events page**: hard refresh (`Cmd+Shift+R`). If it's still stuck, text Zaal.
